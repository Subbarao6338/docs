Knowledge Hub supports various themes and a set of Material You inspired accent colors to make your documentation visually appealing.

## Material You Accent Colors

We have implemented a range of accent colors inspired by Material You's dynamic color palettes.

### Available Colors

| Color | Hex Code | Variable Name |
| :--- | :--- | :--- |
| **Ocean Blue** | `#3884FF` | `--color-primary` |
| **Forest Green** | `#166E3F` | `--color-accent-green` |
| **Royal Purple** | `#6750A4` | `--color-accent-purple` |
| **Sunset Orange** | `#D9410E` | `--color-accent-orange` |
| **Deep Crimson** | `#B3261E` | `--color-accent-red` |

## Theme Customization

You can switch between light and dark modes, and the accent colors will adapt accordingly to maintain accessibility and visual harmony.

### Light Mode
Optimized for readability with high contrast and subtle grid backgrounds.

### Dark Mode
A sleek, modern aesthetic that reduces eye strain in low-light environments.
