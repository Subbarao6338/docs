Google Cloud Platform (GCP) is a suite of cloud computing services that runs on the same infrastructure that Google uses internally for its end-user products.

### Unique Selling Points 🌟

- **Data & Analytics**: Industry-leading BigQuery and AI/ML tools.
- **Kubernetes Roots**: Originator of Kubernetes, providing excellent GKE support.
- **Global Network**: One of the world's largest and most advanced private networks.

### Quick Start Tips

<Tip>
  **Cloud Shell**: Use the built-in Cloud Shell in the console for a pre-configured CLI environment.
</Tip>

<Check>
  **Projects & Billing**: Everything in GCP is organized into Projects. Ensure your billing is linked to the correct project.
</Check>

<Accordion title="Core Data Services">
  - **BigQuery**: Serverless, highly scalable data warehouse.
  - **Pub/Sub**: Messaging and ingestion for event-driven systems.
  - **Dataflow**: Stream and batch data processing.
</Accordion>

### Useful `gcloud` Commands

```bash
# List all projects
gcloud projects list

# Set the current project
gcloud config set project [PROJECT_ID]

# List Compute Engine instances
gcloud compute instances list
```
