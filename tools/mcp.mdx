The Model Context Protocol (MCP) is an open standard that enables developers to build secure, two-way connections between their data sources and AI models.

## Core Concepts

- **MCP Clients**: Applications like Claude Desktop or IDEs that connect to MCP servers.
- **MCP Servers**: Small programs that expose data and tools through the protocol.
- **Resources**: Data sources that can be read by the client.
- **Tools**: Executable functions that the model can call.

## Step-by-Step: Setting Up an MCP Server

1.  **Installation**:
    Ensure you have Node.js or Python installed.
2.  **Creating a Basic Server (Python)**:
    ```python
    from mcp.server.fastmcp import FastMCP

    mcp = FastMCP("MyServer")

    @mcp.tool()
    def get_weather(location: str) -> str:
        return f"The weather in {location} is sunny."

    if __name__ == "__main__":
        mcp.run()
    ```
3.  **Configuring Client**:
    Add your server to the client configuration file (e.g., `claude_desktop_config.json`).
    ```json
    {
      "mcpServers": {
        "myserver": {
          "command": "python",
          "args": ["path/to/server.py"]
        }
      }
    }
    ```
