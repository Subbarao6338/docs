## LaTeX (Mathematical Typesetting)

Knowledge Hub supports LaTeX for rendering complex mathematical equations.

### Examples
- **Inline Math**: $E = mc^2$
- **Block Math**:
$$
\frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$

## YAML (YAML Ain't Markup Language)

YAML is a human-readable data serialization language, commonly used for configuration files.

### Step-by-Step: YAML Structure
1.  **Key-Value Pairs**: `key: value`
2.  **Lists**:
    ```yaml
    items:
      - item1
      - item2
    ```
3.  **Nested Objects**:
    ```yaml
    person:
      name: John
      age: 30
    ```

## Step-by-Step: Using LaTeX in MDX

1.  **Enable LaTeX**: Ensure `"latex": true` is set in your `docs.json` styling configuration.
2.  **Write Equations**: Use single dollar signs (`$`) for inline math and double dollar signs (`$$`) for blocks.
3.  **Verify**: Preview your page to ensure equations render correctly.
